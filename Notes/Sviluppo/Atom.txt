===== Preparazione del progetto di lavoro =====
Per creare l'applicazione abbiamo bisogno di electron, react e material-ui.

Per cui:

git clone  http://github.com/pieroproietti/electron-react-boilerplate fabrica-installer

cd fabrica-installer
npm install material-ui  react-tap-event-plugin react react-dom -s
npm installm

===== Avvio =====

$ npm run hot-server
$ npm run start-hot
